Election system for students designed with simplicity in mind. Integrates with an existing timetable system. Note that the orignal recipient of this project has been annonymised

Built with (backend):

* .NET Framework 4.7.2
* ASP.NET MVC 5
* Entity Framework 6
* Hangfire
* AutoMapper

Built with (frontend):

* Bootstrap 4
* jQuery
* DataTables
